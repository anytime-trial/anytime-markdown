## 詳しい説明（4000文字以内）

### 日本語

Anytime Markdown は、いつでもどこでもマークダウンを快適に編集できるリッチエディタアプリです。

主な機能:
- リッチテキスト編集 - 見出し、太字、リスト、テーブルなどをツールバーから簡単に挿入
- Mermaid ダイアグラム - フローチャート、シーケンス図、ガントチャートをマークダウン内に描画
- PlantUML 対応 - UML図をコードブロックから直接レンダリング
- リアルタイムプレビュー - 編集しながら即座に結果を確認
- ダークモード - 目に優しいダークテーマを標準搭載
- キーボードショートカット - 効率的な編集をサポート
- オフライン対応 - インターネット接続なしでも利用可能

エンジニア、テクニカルライター、学生など、マークダウンを日常的に使うすべての方に最適なエディタです。

### English

Anytime Markdown is a rich editor app that lets you comfortably edit markdown anytime, anywhere.

Key Features:
- Rich Text Editing - Easily insert headings, bold, lists, tables and more from the toolbar
- Mermaid Diagrams - Draw flowcharts, sequence diagrams, and Gantt charts within markdown
- PlantUML Support - Render UML diagrams directly from code blocks
- Real-time Preview - See results instantly as you edit
- Dark Mode - Eye-friendly dark theme included
- Keyboard Shortcuts - Efficient editing support
- Offline Support - Works without internet connection

The perfect editor for engineers, technical writers, students, and anyone who uses markdown daily.
