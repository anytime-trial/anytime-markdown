# Change Log

All notable changes to the "Anytime Git" extension will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/).

## [Unreleased]

### Added

- Initial release
- Repository view with folder open, clone, branch switch, file CRUD, drag-and-drop, Markdown filter
- Changes view with staging, unstaging, commit, push, discard, sync, and change count badge
- Graph view with ASCII commit graph, local/remote indicators, branch/tag decorations
- Timeline view with per-file commit history and commit comparison
- Markdown compare mode integration with Anytime Markdown (optional dependency)
- Fallback to VS Code standard diff editor when Anytime Markdown is not installed
